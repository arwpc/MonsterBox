    Coffin Door Wiring Note

    The coffin door actuator's direction pin polarity is inverted compared to Orlok's actuators. invertDirection: true in parts.json swaps extend/retract at the software layer so API commands match physical movement: - API "extend" (open) → hardware sends dir=1 (HIGH) - API "retract" (close) → hardware sends dir=0 (LOW)
