# Authorization

## No Role-Based Access Control

MonsterBox does **not** implement RBAC, user roles, or permission systems. All users on the MonsterNet local network have full access to all features:

- View and control all characters
- Create, edit, and delete scenes and poses
- Control hardware (servos, motors, LEDs, sensors)
- Manage AI settings and configuration
- Access system diagnostics

## Single-Operator Design

MonsterBox is designed as a single-operator system for controlling Halloween animatronics. The operator has full control over all characters and hardware from the Dashboard.

For details on the overall security approach, see [Security Model](authentication.md).
