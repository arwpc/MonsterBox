# Orlok - Character Sheet

## Overview
Orlok is the primary animatronic character in the MonsterBox system, inspired by the classic Nosferatu vampire.

## Hardware Components
- Servo-controlled jaw for speech animation
- LED eyes
- Head tracking servos
- Speaker for TTS audio output

## Configuration
Character data is stored in `data/character-3/` including parts, poses, scenes, and super-powers configuration.
