# MonsterBox

MonsterBox is an RPi4B-based animatronic character control platform built with Node.js, Express, and EJS templates.

## Features

- Multi-character animatronic control (servos, motors, LEDs, linear actuators)
- Animation Studio for scene and pose creation
- AI-powered conversation with ElevenLabs TTS/STT integration
- Real-time jaw animation synchronized to audio
- Head tracking via webcam
- Audio library management
- Hardware calibration system
