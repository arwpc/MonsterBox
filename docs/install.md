# Installation

## Prerequisites

- Raspberry Pi 4B (4GB+ RAM recommended)
- Node.js 20.x
- Python 3.x
- PipeWire audio system

## Setup

1. Clone the repository
2. Run `npm install`
3. Run `bash setup-monsterbox.sh` for full system setup
4. Start with `npm start` or `sudo systemctl start monsterbox`

See the [Animatronic Setup Guide](setup/ANIMATRONIC-SETUP-GUIDE.md) for detailed hardware configuration.
