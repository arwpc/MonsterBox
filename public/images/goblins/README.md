# Goblin Avatar Images

Place goblin avatar images here. Supported formats: PNG, JPG, GIF

## Default Images:
- `goblin1.png` - Default for Goblin 1
- `goblin2.png` - Default for Goblin 2
- `goblin3.png` - Default for Goblin 3
- `goblin4.png` - Default for Goblin 4
- `goblin5.png` - Default for Goblin 5
- `default.png` - Fallback image

## Custom Images:
Upload custom goblin images through the Edit Settings modal.

Images should be square (recommended: 256x256px) for best results.
